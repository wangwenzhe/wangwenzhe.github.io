注意

使用范围

百度地图 API 可让您在接受使用条款约束的情况下，在您的网站上显示百度地图图片；进行地点搜索、路线查询和交通流量显示等操作。 您只可使用在百度地图 API 文档中所列明开放的 API 功能来对 API 相关服务数据的结果进行展示，不得直接存取、使用内部数据、图片、程序、模块或是任何其他百度地图的服务或功能。在接受使用条款约束的情况下，您可以在向最终用户提供其他信息的同时，使用 API 获得 API 相关服务数据。必须保留百度版权信息。

商业应用与收费情况

百度地图 API 以开放形式提供给开发者，如果满足您的需求且不违背 “百度地图 API 使用条款” 的情况下，可以直接使用。针对商业应用场景，建议申请百度地图商业授权服务，授权通过后可获得权威纸质盖章版授权证书，具备法律效力。

地图运营资质

地图运营资质

百度地图 API 拥有国家测绘局颁发的互联网地图运营资质，用户可放心使用百度地图 API 来制作自己的网站地图应用。

地图审图号（移动端）：GS（2017）2638

地图审图号（PC 端）：GS (2018) 5572

测绘资质证书编号：甲测资质 1100930

互联网信息服务增值电信业务经营许可证编号：ZS180036 但这并不代表用户可随意运营地图服务。建议用户参考国家测绘局颁发的具体条款： http://chzt.sbsm.gov.cn/article/zxgz/hlwdtfw/ygzchbz/201103/20110300081026.shtml

是否有商业版与大众版之分？

百度地图 API 收费服务可提供更高的服务权限和技术支持。

是否可以去掉百度地图左下角百度版权 logo？

不可以。任何使用百度地图 API 调用地图服务应用必须保留百度版权信息。相关参看【1.2 使用范围】。

坐标体系

坐标体系是否遵循国家对地理信息保密要求？

百度对外接口的坐标系，都是经过国家测绘局加密处理，符合国家测绘局对地理信息保密要求。

百度采用何种坐标体系？

百度采用 BD09 坐标系，BD09 坐标系是在 GCJ02 坐标系基础上再次加密形成。其中 bd09ll 表示百度经纬度坐标，bd09mc 表示百度墨卡托米制坐标。

具体信息可参考：http://lbsyun.baidu.com/index.php?title=coordinate

百度坐标为何有偏移？

国际经纬度坐标标准为 WGS-84, 国内必须至少使用国测局制定的 GCJ-02, 对地理位置进行首次加密。百度坐标在此基础上，进行了 BD-09 二次加密措施，更加保护了个人隐私。百度对外接口的坐标系并不是 GPS 采集的真实经纬度，需要通过坐标转换接口进行转换。

如何从其他体系的坐标迁移到百度坐标？

开发者可以使用坐标转换接口进行转换。JavaScript API 、Android SDK、iOS SDK 的开发用户可直接调用相应方法进行转换。

官方反馈通道

开发过程中遇到问题如何解决？

如您在开发过程中遇到问题，请访问反馈与帮助浏览常见问题或创建新的问题，百度地图工程师将会对您的问题进行回复与解答。

地图数据有错误，如何反馈？

若您发现百度地图的数据（如地点、名称、电话等信息，公交、道路、三维图等）有错误，请登录百度地图数据错误反馈网址：http://tousu.baidu.com/map/add 反馈，将有专人处理，地图数据一季度更新一次。
