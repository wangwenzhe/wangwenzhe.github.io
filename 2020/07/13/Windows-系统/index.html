Win7 PPTP VPN 连接操作步骤

http://nic.upc.edu.cn/2018/0315/c7809a137813/page.htm

一、配置 PPTP VPN 步骤：

  1. 右键右下角有线网络图标（或无线网络图标），点击 打开网络与共享中心（或右键桌面上的 网络，点击 属性，进入 网络与共享中心 页面)，并点击 设置新的连接或网络，如下图所示：

  1. 选择 连接到工作区，点击 下一步。如下图所示：

  1. 选择 使用我的 Internet 连接（VPN）(I)，如下图所示：

  1. 在 Internet 地址中输入 VPN 服务器的地址：vpn.upc.edu.cn，下面三个选项选择 “现在不连接；仅进行设置以便稍后连接”，点下一步，如下图所示：

  1. 在弹出的对话框中，不输入用户名和密码，直接点创建，如下图所示：

  1. 在跳转到的页面点 关闭。如下图所示：

二、连接 VPN 步骤：

  1. 找到 “网络和共享中心”，点击左侧 “ 更改适配器设置 ”，如下图所示：

  1. 右键 VPN 连接，打开属性选项卡中安全选项，在 VPN 类型中，选择 “点对点隧道协议（PPTP）”，在数据加密中，选择 “可选加密（没有加密也可以连接）”，在允许使用这些协议中，勾选 “未加密的密码（PAP）” 和 “质询握手身份验证协议（CHAP）”，如下图所示：

  1. 双击 VPN 连接，在弹出窗口，输入 VPN 用户名和密码，点连接，完成 VPN 连接。如下图所示：

win7 系统怎么从家庭普通版升级到旗舰版

控制面板 ——> 系统和安全 ——> Windows Anytime Upgrade ——> 输入升级密钥 ——> 输入 RHTBY-VWY6D-QJRJ9-JGQ3X-Q2289 ——> 耐心等待系统的升级完成

修改 hosts 立即生效

win10 操作中心开关呈灰色无法打开

  1. 运行 gpedit.msc
  2. 用户配置 - 管理模板 - “开始”菜单和任务栏-右侧设置中「删除通知和操作中心」-双击打开新窗口-来回切换至「未配置」-确定
  3. 设置-任务栏通知区域-打开或关闭系统图标-操作中心

若提示因为计算机中丢失MSVCR120.dll

解决：下载 msvcr120.dll，放到 C:\Windows\System32\msvcr120.dll

无法删除文件或文件夹，提示无权限

  1. 当前账号切换为 administrator，没有 administrator 的，需要先激活

  2. 修改文件夹所有者为administrator

用户

win10 创建管理员账户（非 administrator）

控制面板-用户账户-管理其他账户-在电脑设置中添加新用户-将其他人添加到这台电脑-本地用户和组（本地）-用户-右键新用户-创建-选择用户右键-属性-隶属于-Users删除-添加-输入对象名称来选择-administrators-确定

Win7 管理员 Administrator 用户启用、禁用

  • 启用 Administrator

    net user administrator /active:yes

  • 禁用 Administrator

    net user administrator /active:no

  • 图形界面操作

    计算机管理-系统工具-本地用户和组-用户

设备

修改鼠标滚动方向

输入法

搜狗输入法中文状态下无法输入中文

ctrl+shift+E 切换下即可「属性设置-高级-系统功能快捷键-英文输入法」

远程

Windows 远程桌面 出现 身份验证错误，要求的函数不受支持

解决方法：运行 gpedit.msc 本地组策略：计算机配置 > 管理模板 > 系统 > 凭据分配 > 加密 Oracle 修正，选择启用并选择易受攻击

如果没有本地组策略，可能是因为当前 Windows 是家庭版，例如 Windows 7 家庭版，参考上面『win7 系统怎么从家庭普通版升级到旗舰版』，升级后就可以进行后面的操作了

Mac 通过 AnyDesk 远程连接Windows，键盘无法输入密码

把 Mac 输入法暂调为英文键盘即可

模拟器

运行安卓模拟器蓝屏解决方案

关闭 Hyper-V 和 Windows 沙盒，运行以下命令，重启电脑

中文乱码

使用 notepad++ 打开，编码-转为 UTF-8 编码

程序和功能

‘telnet’ 不是内部或外部命令，也不是可运行的程序或批处理文件

解决方法：控制面板——> 程序和功能——> 启动或关闭 Windows 功能——> 勾选 Telnet

NET Framework

若要运行此应用程序，您必须首先安装.NET Framework 的以下版本之一

解决方法：根据提示安装.net FrameWork 即可 ，下载Microsoft .NET Framework 4（独立安装程序）
