Win10-FTP开启

Win10 FTP开启

https://blog.csdn.net/qq_34610293/article/details/79210539

1. 开启 FTP

控制面板——程序和功能——启用或关闭 Windows 功能——Internet Infomation Services

2. 添加 FTP站点并配置「局部」

  1. 搜索 IIS 打开 IIS

  2. 网站右键添加 FTP 站点，输入站点名称和作为 FTP 的目录

  3. 绑定和 SSL 设置：IP 地址下拉选择本机 IP，选择无 SSL

  4. 身份验证和授权信息：根据需要选择身份验证，授权与权限

3. 防火墙设置

  1. 控制面板 > 系统和安全 > 防火墙 > 允许应用或功能通过 Window 防火墙 >点击更改设置，勾选 FTP 服务器 & 专用 & 公用
  2. 点击允许其它应用，选择 C:\Windows\System32\svchost.exe 然后添加，最后确定

4. 访问

打开资源处理器，输入 ftp:// IP

windows server 2008 R2 开启 ftp 服务

https://blog.csdn.net/XiaoXiao_RenHe/article/details/76067468

1. 开启 FTP 服务

  1. 开始 —— 管理工具——服务器管理器
  2. 角色——添加角色
    1. Web 服务器（IIS）
    2. FTP 发布服务
    3. 安装

2. 全局配置 FTP

  1. 点击服务器证书-创建自签名证书
  2. FTP SSL 设置——允许 SSL 连接——应用
  3. FTP 身份验证——基本身份验证——右键-启用
  4. FTP 授权规则——添加允许规则——指定的角色或用户组——Administrators——读取、写入

3. 添加 FTP 站点

网站——右键-添加 FTP 站点

4. 重启

5. 防火墙入站规则添加 21 端口

问题

  1. windows2003 和 windows2008 服务器端安装 IIS 自带的 FTP 功能，如果防火墙关闭才可以通过 FTP 正常访问，打开 FTP 则不能访问，解决办法如下：

    windows2003 服务器端设置

    控制面板》系统和安全》Windows 防火墙》允许的程序”，在 “允许另一个程序” 中添加 “C:\WINDOWS\system32\inetsrv\inetinfo.exe” 这个程序，添加完成即可

    windows2008 服务器端设置

    控制面板》系统和安全》Windows 防火墙》允许的程序”，在 “允许另一个程序” 中添加 “C:\Windows\System32\svchost.exe” 这个程序，添加完成即可
