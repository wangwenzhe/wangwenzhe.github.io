MySQL-Workbench

发表于2021-09-08|更新于2025-06-15|数据库MySQL-Workbench
|字数总计:105|阅读时长:1分钟|阅读量:

导出

  1. 打开 workbench

  2. 左侧，管理视图， 点击 Data Export

  3. 选择数据库，选择 dump Structure and Data 导出数据结构和数据，选择导出文件位置，也可以勾选 Include Create Schema，包含创建数据库语句

  4. 切换页签到 Export Progress，点击 Start Export

导入

  1. 打开 workbench
  2. 点击 File- Run SQL Script，选择 SQL 文件
  3. 选择目标数据库，选择字符集编码 utf8mb4
  4. 点击 Run
