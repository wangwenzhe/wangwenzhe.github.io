说明

  • Javaassist 就是一个用来处理 Java 字节码的类库
  • Java 字节码以二进制的形式存储在 .class 文件中，每一个 .class 文件包含一个 Java 类或接口
  • Javaassist 可以在一个已经编译好的类中添加新的方法，或者是修改已有的方法
  • Javaassist 同时也可以去生成一个新的类对象，通过完全手动的方式
