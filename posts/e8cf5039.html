概述

文本内容比较工具

下载

链接: https://pan.baidu.com/s/1ZNPQ8DCKEXnr2mFJAKZF6g 密码: sal6

比较 class 或 jar 文件

  1. 下载支持 Java class 转换配置文件http://www.scootersoftware.com/download.php?zz=kb_moreformats_win

    1. Select your version of Beyond Compare
    2. Select your platform
    3. Select and download one or more file formats from the list

  2. 打开Compare，点击工具（Tools）–> 导入设置（Import Setting）–> 选择下载的文件BCFormats.bcpkg，导入即可

  3. 重新打开Compare，比较两个 jar
