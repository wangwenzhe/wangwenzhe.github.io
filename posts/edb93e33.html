centos 离线安装docker-ce

目前越来越多的应用采用docker部署管理，使快速的部署集成变得更容易。但笔者在安装docker时，遇到一个问题，目标服务器无法联网，这就要求我们能够采用离线的方式安装docker。下面就如何离线安装docker，列出了实施步骤，与大家一起分享。

1. 系统环境

Centos7 Linux version 3.10.0-327.10.1.el7.x86_64

2. docker-ce****版本

18.03.0.ce-1.el7.centos.x86_64

3. 在可以联网的机器上制作docker本地安装源

3.1 安装一个虚拟机，然后在虚拟机里面安装centos7（当然，也可以一台可以联网安装了centos7系统的主机）

3.2 创建离线包存放目录,并赋予读写权限

  1. # mkdir -p /home/docker/local
  2. # chmod -R 777 /home/docker/local

3.3 创建docker的yum repository

3.3 下载离线docker-ce依赖包

注意，下面指令仅仅下载离线包到指定目录。后面要用到。

  1. yum install *–downloadonly –downloaddir=/home/docker/local yum-utils *
  2. device-mapper-persistent-data \
  3. lvm2 createrepo

3.4安装docker-ce依赖包及createrepo

  1. yum install yum-utils \
  2. device-mapper-persistent-data \
  3. lvm2 createrepo

3.5 配置docker repository

  1. yum-config-manager \
  2. –add-repo \
  3. https://download.docker.com/linux/centos/docker-ce.repo

3.6 更新yum源索引

yum makecache fast

3.7下载离线docker-ce安装包

yum install –downloadonly –downloaddir=/home/docker/local docker-ce

3.8 获取安装docker需要的密匙

查看密匙文件url

  1. [jerry@localhost ~]$ more /etc/yum.repos.d/docker-ce.repo
  2. [docker-ce-stable]
  3. name=Docker CE Stable - $basearch
  4. baseurl=https://download.docker.com/linux/centos/7/$basearch/stable
  5. enabled=1
  6. gpgcheck=1
  7. gpgkey=https://download.docker.com/linux/centos/gpg
  9. [docker-ce-stable-debuginfo]
  10. name=Docker CE Stable - Debuginfo $basearch
  11. baseurl=https://download.docker.com/linux/centos/7/debug-$basearch/stable
  12. enabled=0
  13. gpgcheck=1
  14. gpgkey=https://download.docker.com/linux/centos/gpg

下载密匙文件(存放到/home/docker/local下)

  1. cd /home/docker/local
  2. wget https://download.docker.com/linux/centos/gpg

3.9 初始化安装源repodata库

  1. # createrepo -pdo /home/docker/local /home/docker/local
  2. # createrepo –update /home/docker/local

3.10 打包导出安装源

# cd /home/docker/local# tar -zcvf docker-local.tar.gz *

docker-local.tar.gz这个文件，就是我们制作好的离线安装源。

**4.**将docker-local.tar.gz copy到无法联网的centos7 服务器上(比如/home/docker/local目录下)，并解压

  1. # mkdir -p /home/docker/local （如果不存在此目录，则创建,然后copy docker-local.tar.gz到此目录）
  2. # cd /home/docker/local
  3. # tar -zxvf docker-local.tar.gz

5 安装createrepo

  1. # cd /home/docker/local

  2. # rpm -ivh createrepo-0.9.9-28.el7.noarch.rpm (这个rpm安装包就是上面我们离线下载的安装包)

6 备份安装源

  1. # cd /etc/yum.repos.d/
  2. # mkdir repobak
  3. # mv * repobak/

7 配置离线安装源

  1. # vi /etc/yum.repos.d/docker-ce-local.repo

8 构建本地安装源

# createrepo -d /home/docker/local/repodata# yum clean all (清除缓存)

# yum makecache (创建缓存)

9 检查本地库是否制作成功

# yum repolist

repo id repo

name status

docker docker

local 38

repolist: 38

10. 安装docker-ce

# yum install docker-ce

**11.**检查是否安装成功

  1. # systemctl start docker
  2. # docker version

12. 创建docker用户组

12.1 创建docker用户组

# groupadd docker

12.2 将指定用户（比如ittest）加入docker用户组

# usermod -aG docker ittest

12.3 重启docker

# systemctl restart docker
