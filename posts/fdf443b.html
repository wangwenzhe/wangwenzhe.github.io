设置固定ip

Ubuntu 配置 Netplan 永久修改路由默认网关及错误处理 - 新动弹计划 (cellmean.com)

要在 VMware 中运行的 Ubuntu 22.04 上设置固定的 IP 地址（当使用桥接模式时），你需要按照以下步骤操作来配置网络：

  1. 打开终端：
    打开你的 Ubuntu 虚拟机并启动一个终端窗口。

  2. 编辑网络接口配置文件：
    找到网络接口配置文件。通常情况下，Ubuntu 使用 netplan 来管理网络配置。你可以通过编辑 /etc/netplan/ 目录下的配置文件来设置静态 IP 地址。默认情况下，可能会有一个名为 00-installer-config.yaml 的文件

    使用文本编辑器打开这个文件，例如使用 nano 或 vi：

  3. 修改配置文件：
    在配置文件中添加或修改网络接口的配置以设置静态 IP 地址。下面是一个例子，假设你想要给 eth0 接口分配一个静态 IP 地址 192.168.1.100，子网掩码 255.255.255.0，网关 192.168.1.1，并且设置 DNS 服务器为 8.8.8.8 和 8.8.4.4：
